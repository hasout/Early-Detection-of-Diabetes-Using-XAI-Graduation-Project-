# Early Detection of Diabetes Using Explainable AI

## Overview
This graduation project focuses on developing an Explainable AI (XAI) model for early diabetes detection. The model was built using machine learning techniques to achieve high prediction performance while providing interpretable explanations for healthcare professionals.

## Technologies
- Python
- XGBoost
- SHAP
- Pandas
- NumPy
- Scikit-learn

## Features
- Early diabetes prediction
- Explainable AI using SHAP
- Cost-sensitive learning
- Threshold optimization
- High recall performance

## Results
- Achieved 96.41% Recall
- Improved prediction interpretability
